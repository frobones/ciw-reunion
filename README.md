# CIW Annual Reunion Website

A recreation of the CIW Annual Reunion Dinner website, designed for hosting on Squarespace.

## Project Structure

```
CIW/
├── index.html                              # Complete standalone HTML
├── css/
│   ├── styles.css                          # Full stylesheet
│   └── squarespace-custom.css              # Squarespace-specific CSS
├── js/
│   └── main.js                             # Navigation and animations
├── assets/                                 # Place images here
└── docs/
    └── squarespace-implementation-guide.md # Detailed implementation guide
```

## Quick Start

### Preview Locally

Open `index.html` in a web browser to preview the complete website design.

### Implement on Squarespace

1. **Choose a Template**: Brine, Bedford, or Montauk work well
2. **Build Page Structure**: Use native Squarespace blocks for each section
3. **Add Custom CSS**: Paste `css/squarespace-custom.css` into Design > Custom CSS
4. **Add Fonts**: Add Google Fonts link in Settings > Advanced > Code Injection > Header
5. **Upload Photos**: Add reunion photos to the gallery section

See `docs/squarespace-implementation-guide.md` for detailed instructions.

## Design Features

- Modern, celebratory design with gold and navy color scheme
- Fully responsive layout for mobile and desktop
- Animated hero section with staggered text reveal
- Timeline-style event schedule
- Grid-based photo gallery with hover effects
- Team/committee member showcase
- Payment options with clear instructions

## Typography

- **Display Font**: Cormorant Garamond (headings)
- **Body Font**: Montserrat (paragraphs, navigation)
